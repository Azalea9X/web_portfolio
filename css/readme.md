<h1>Readme</h1>
<p>This is where you will find the different CSS files from my files. The important files that you will find with Tailwind are the tailwind.config.js, the packages, the input file and the output.min.css.</p>
<p>Output.min.css relies on different input files on my files which I have used. I have taken an example from Tailwind's website, and, they do say themselves that you are allowed to take the file and use it for a demonstration. For my purposes, I am not using it as a demonstration, I am using it to develop CSS for the output.min.css file. The way JIT works is that it takes the CSS that you input it and it outputs it directly and only takes what it needs. It's very efficient, but, you need to remember to do it based off the inputs, otherwise, it will not work. Tailwind is a work in progress, but, it is lightweight, and, less opinionated than Bootstrap, which, to me gives me an advantage with it.</P>
