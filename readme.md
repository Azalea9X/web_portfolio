<h1>Important notes</h1>
<p>As you look over my portfolio, I will note that most of my PHP examples either come from LinkedIn Learning or PHP. I'm more familiar with the JavaScript language, but, I do think that it is important to know more than one language, and, I have been told that if you really want to be able to customize WordPress and Drupal, you need to know the PHP language.</p>
<P>Anything that is labeled with front-end will only use regular front-end tools. That means doing anything that requires an authorization is silly using the front-end, because, without a back-end to go with it, it will not accept the front-end. So, you could think of my front-end apps as more of a proof of concept, because, it needs a database to save the information that you put on it, and, the backend apps are more of the way that things work. Front-end apps are essentially incomplete proof of concepts.</p>
